# 北京航空航天大学 VI 标志组合

本仓库为北航（BUAA）VI 视觉识别系统中的标志组合 PNG 导出文件，所有颜色严格遵循官方 VI 品牌色彩规范。

## 颜色规范

| 颜色 | C | M | Y | K | 说明 |
|------|---|---|---|---|------|
| 🟦 北航蓝 | 100 | 60 | 0 | 0 | 主识别色 |
| 🔵 星空蓝 | 100 | 80 | 10 | 0 | 辅助色 |
| 🌐 天空蓝 | 100 | 30 | 0 | 0 | 辅助色 |
| 🔴 中国红 | 15 | 100 | 90 | 10 | 辅助色 |
| ⚫ 品质灰 | 0 | 0 | 0 | 60 | 辅助色 |
| 🖤 墨稿 | 0 | 0 | 0 | 100 | 单色印刷 |
| 🤍 反白稿 | 0 | 0 | 0 | 0 | 白字透明底，用于深色背景 |
| 🥇 专色金 | 20 | 42 | 66 | 0 | 烫金工艺 |
| 🥈 专色银 | 21 | 15 | 15 | 0 | 烫银工艺 |

## 导出规格

- **格式**：PNG-24，透明背景
- **分辨率**：300 PPI
- **颜色模式**：CMYK
- **导出工具**：Adobe Illustrator 2025 + ExtendScript
- **边缘处理**：抗锯齿（图稿优化）

## 使用说明

- **墨稿**（K100）用于黑白印刷、水印等单色场景
- **反白稿**为纯白字＋透明底，可叠加在任意深色背景上使用
- **专色金 / 专色银**用于烫金烫银等特殊印刷工艺
- 所有 PNG 支持直接用于 Office 文档、网页、PPT、视频包装等数字媒介
